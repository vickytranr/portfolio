In 2020, I participated in Ubisoft's annual student competition, Ubisoft NEXT, and the following entry was a finalist in the UI category.

Overview

The project called for the creation of a UI design for an existing Ubisoft IP. I chose to create designs for Farcry Primal. The brief essentially calls for:

  • a Loadout Menu UI
  • a Gameplay HUD UI
  • matching iconography representing damage, fire rate, ammo, and durability
  • motion concepts
